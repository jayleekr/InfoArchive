# Shortcut

## Must-Know Shortcut

 유용한 단축키는 언제든 공유해주세요!!!! 돕고살아요!

| Key-Binding        | Function        | Description                            |
| ------------------ | --------------- | -------------------------------------- |
| F1 Or Ctrl+Shift+P | command palette | VSCode 내 모든 기능을 검색가능하다     |
| Ctrl + `           | Open Terminal   | 터미널을 빠르게 띄울수 있다            |
| Ctrl + Shift +  I  | Markdown 표정리 | Markdown 양식의 표를 이쁘게 만들어준다 |
